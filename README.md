# WordBookApp

## PURPOSE OFF THE APPLICATION :
### The purpose of this application is to create a foreign language vocabulary book electronically and to visually support the learning process.

## APP'S TECHNOLOGIES : 
#### • MVVM
#### • Hilt-Dagger
#### • Room DB
#### • Coroutines
#### • Live Data
#### • Retrofit
#### • Navigation Component
#### • Glide
#### • ViewBinding
